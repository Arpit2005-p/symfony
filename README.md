# Symfony User Management System

## Features
- Login & Registration with password hashing
- CRUD operations using Doctrine ORM
- Entity relationships & lifecycle callbacks
- Admin Panel (Sonata Admin)
- Role-based access control
- User CRUD operations
- Secure authentication

## Tech Stack
- Symfony 6.4
- PHP
- MySQL
- Doctrine ORM
- Sonata Admin
